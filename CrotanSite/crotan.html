What is Croton?

A multiparadigm, dynamically typed, general-purpose, futuristic, verbose, HIGH-LEVEL programming language that stands for:

Scalablity

Even though Croton is a verbose language, it's syntax and controllability allows it to be used at the greatest extent. Small applications can be extended to make fully-fledged projects without breaking any code.

Uniqueness

Unlike most languages, Croton features a whole new syntax that differs from the traditional C-style-syntax. This allows Croton code to be distinguishable in polyglot programming. It also uses various algorithms and methods to get your program to work the way you want it to.

Control

This is where this programming language really shines. Croton allows you to control how your code works. Meaning that you can choose how a function or variable interacts with other programatic phenomenons such as scope! As the programmer is in control of these phenomenons, the program is easy to understand and control

Your language, your choice

The Croton Language execute anything on its own, instead it only does what you tell it to. You can disable language features that seem useless to you and enable the ones that sound appealing. This essentially means that you can control what the interpreter parses and what it ignores.

Build it in Croton

Even though Croton is not optomised for performance critical tasks, you can still:

Integrate with NodeJS & BunJS

Croton is build in JavaScript and therefore it has direct access to all JS modules. It can do everything JS can do! All you need to do is direcly import a JavaScript file and use it any way you want!

Make Domain Specific Language!

Croton's syntax is entirely extensible. You can write your own rules using the parser API and can extend the syntax to make your own Domain Specific Language!

Make websites!

Use Croton server to use client-side JS capibilites. All you need to do is attain a copy of Croton server which compiles code into JavaScript and you are good to go!
